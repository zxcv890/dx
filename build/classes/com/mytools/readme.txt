这是一个工具类，做为保存，以后会按功能不同加以文件夹进行区分，每个文件夹下再会有一个readme进行说明该文件夹下的类都是什么工具
GetTimeTool 里面保存了三个方法，以当前的时间来分别修改前几天、小时、分钟。功能需要再扩展一下，可以是当前时间，也可以是以某时间为基点
MyQRCode 是生成二维码的工具类，需要引入jar包。生成过程是得到图片，由绘画类在图片上选定位置，指定大小，生成一个带有特定信息的二维码加原图片，再以流的形式转到页面
MyDes 是des加密工具，里面的方法经过了lsp项目的考验。通过固定的key值来进行加密解密。
MyPage 里面的方法大多基于jfinal ，保留下来为了以后方便查看算法。目前有个分页的html拼接可用。
CheckMobile 是判断是否是用手机或平板登录
